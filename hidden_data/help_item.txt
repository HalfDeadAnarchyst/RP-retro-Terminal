 NAME
		ITEM
		
 SYNOPSIS
		ITEM [OPTION]
		
 DESCRIPTION
		ITEM - Displays summarized information about transponders in the zone or of the owner

		ZONE
			Displays all summarized transponder entries in the selected ZONE
		
		OWNER
			Displays all summarized transponder entries of the selected OWNER
		
		VERSION
			Output version information
			
 EXAMPLES
		ITEM ZONE CARGO
			Output all responded transponders in the CARGO bay

		ITEM OWNER SHIPNAME
			Output all registered transponders of the OWNER
			
 NOTE
		PING is used for precised triangulation of item with known name
